Comptes bancaires¶

Dans cet écran, vous avez la possibilité d’enregistrer vos différents comptes bancaires que vous possédez. Pour chacun, vous pouvez saisir l’intégralité des informations figurant sur un RIB. Cela vous permettra d’éditer un résumé complet de vos dépôts de chèques.
