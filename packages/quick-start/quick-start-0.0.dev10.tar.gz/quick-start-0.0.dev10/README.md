# QuickStart
> Cria um projeto baseado em um modelo

## Instalação

Linux:

```sh
 $ python setup.py install --user
 $ ~/.local/bin/quickstart_installer
```


## Exemplo de uso

``` sh
$ pyinit nome_do_projeto
```

## Configuração para Desenvolvimento

```sh
python setup.py develop
```
