    ReduxBridge

    Extends:

    WebSocketBridge → ReduxBridge

    Bridge between Channels and Redux. By default dispatches actions received from channels to the redux store.

    Example:

    const reduxBridge = new ReduxBridge();
reduxBridge.connect();
reduxBridge.listen(store);
